  if (n / 10 < 1) {
    return n;

    n

}

- Solution for [Array.diff][5]:

```js
function array_diff(a, b) {
  let c = [];
  a.map(value => {
    if (!b.includes(value)) c.push(value);
  });
  return c;
}
```
